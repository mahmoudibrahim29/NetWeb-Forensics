# Network & Web Application Forensics Case Study

## Overview
This project simulates a real-world cyber incident involving:
- Network reconnaissance
- Port scanning
- DNS exfiltration
- SQL Injection
- Web shell upload

The investigation follows a structured SOC workflow:
1. Case Initialization
2. Evidence Preservation
3. Initial Assessment
4. Deep Analysis
5. Event Correlation
6. Final Reporting

## Tools Used
- Wireshark
- Zeek
- Splunk (SIEM)
- Nmap
- SQLMap
- Apache Logs

## Skills Demonstrated
- PCAP analysis
- Log correlation
- Web attack detection
- IOC extraction
- Timeline reconstruction
- Threat profiling
